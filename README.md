# VR-Realistic-Bookshelf

## Getting Started

### Prerequisites

* VR-ready PC
* [Unreal Engine 4](https://www.unrealengine.com/)
* PC-VR Headset & Controllers

## Built with

* [Unreal 4.4.13](https://www.unrealengine.com/) - Game Engine for VR
* HTC Vive - Headset and controllers used
